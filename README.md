# 旅のしおり 📖

地図でエリアを選んで、旅館・温泉・観光スポットを探し、
乗換案内つきの1日プランを「旅のしおり」風にまとめられるWebアプリです。

## できること
- 🗺 地図でエリアを選んでスポット検索（旅館・温泉・観光・グルメ）
- 🏠 各スポットの公式HP・Googleマップへのリンク
- 📋 切符風カードで1日のプランづくり（時刻・メモ編集）
- 🚃 予定間にYahoo!乗換案内・Googleマップ経路リンクを自動生成
- 📖 和紙デザインの「しおり」表示 → スクショ / PDF保存

## 技術メモ
- 単一HTMLファイル（フレームワーク不要）
- 地図：Leaflet + OpenStreetMap
- スポット検索：Overpass API
- データ保存：localStorage（DB不要）
